# Handy App

منصة خدمات منزلية تربط العميل بالصنايعي داخل نفس المنطقة.

## النسخة الأولى

- تطبيق Flutter واحد يعمل بحساب عميل أو صنايعي.
- Android أولًا.
- لا دفع إلكتروني.
- لا تتبع GPS.
- لا توثيق إلكتروني مدفوع.
- الدفع كاش بعد تنفيذ الخدمة.
- مطابقة الطلبات حسب التخصص والمنطقة.

## طريقة التنفيذ

1. إنشاء حسابات العميل والصنايعي فعليًا عبر Supabase.
2. إكمال تدفق طلب الخدمة من البداية للنهاية.
3. ربط الطلبات والعروض والصور بقاعدة البيانات.
4. إضافة لوحة إدارة بسيطة.
5. اختبار مغلق في منطقتين وتخصصين.
